- This app shows a list of all Genres. 
  Resource: /genre/movie/list
- You select a genre and see a list of movies of the corresponding genre.
  Resource: /genre/{genre_id}/movies
- You select a movie and see the poster and overview of the movie.
  Resource: /movie/{movie_id}
